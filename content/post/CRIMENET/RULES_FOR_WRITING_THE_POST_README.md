# Rules for Writing the Post

The rules (set in stone):

1. **No em dashes.** Ever.
2. **Write concisely.** Don't pad.
3. **Don't invent.** Only state what the data and project information support.
4. **Don't extrapolate.** Stick to what's documented.
5. **Feynman style.** First principles, clear explanations, no jargon for its own sake.
6. **Only write things you are 100% sure of.** If the data does not explicitly confirm it, do not say it. Do not assume. Do not guess. Do not add color.
