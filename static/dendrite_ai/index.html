How does Twisted Dendrite work?

This is a hybrid recommendation engine that guarantees logical connections while allowing creative freedom.

1. The Base Layer (Last.fm): For every artist in the map, the engine queries the Last.fm graph API to find artists that are statistically similar based on millions of listeners.

2. The Twist (DeepSeek LLM): If you enter a "Vibe", the engine takes the Last.fm recommendations and feeds them into an AI. It asks the AI to swap out those artists for others that share the same musical DNA, but heavily lean into your requested vibe.

The result is a completely unique, context-aware visual map.
